## 🎯 Objetivo do Projeto

Criar um material de consulta visualmente atraente e bem estruturado, que sirva tanto para revisão pessoal quanto para auxiliar outros estudantes que estão iniciando no mundo da segurança de redes.

- **Foco:** Clareza, organização e identidade visual "cyberpunk/tech".
- **Tecnologias:** HTML5, CSS3, Bootstrap, JavaScript e amor pelo conhecimento.
